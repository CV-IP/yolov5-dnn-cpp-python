# yolov5-dnn-cpp-py
yolov5s,yolov5l,yolov5m,yolov5x的onnx文件在百度云盘下载，链接是
链接：https://pan.baidu.com/s/1d67LUlOoPFQy0MV39gpJiw 
提取码：bayj 

python版本的主程序是main_yolov5.py，C++版本的主程序是main_yolo.cpp
运行整套程序只需要安装opencv库(4.0以上版本的)，彻底摆脱对深度学习框架的依赖
